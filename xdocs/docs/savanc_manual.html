Engaging Savan/C Module

Install Savan module into <axis2 deploy folder>/modules folder.

In the services xml file for the service which require Savan enabled add the entry <module ref="Savan"/> If you need all services in the engine Savan enabled add the above entry into axis2.xml.

You can use samples in the samples folder for testing.
